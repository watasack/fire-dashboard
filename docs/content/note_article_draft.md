# 育休を取るとFIREが何年遅れるか、1,000通りの未来で計算してみた

※ FIRE到達年齢・資産額は実際のシミュレーション結果。FIRE成功確率はモンテカルロ推計値（参考）。

---

「育休は長めに取りたいと思ってて〜」

妻がそう言ったとき、正直すぐに賛成できなかった。
共働きでFIREを目指していると、育休という言葉がどうしても「収入の穴」に見えてしまう。

頭では「もちろん、しっかり納得いく期間取って欲しい」と思っている。でも頭の中では別の計算が走る。育休1年分の収入減。時短勤務への移行。保育料。FIREの時期がずれていく〜！

そこで、「ちゃんと計算してから答える」ことにしました。

---

## 前提条件

うちの数字そのままです。

| 項目 | 数値 |
|------|------|
| 夫婦の手取り月収合計 | 75万円（夫35万・妻40万） |
| 生活費 | 28万円/月 |
| 住宅ローン | 10万円/月（2060年末完済） |
| 支出合計 | 38万円/月 |
| 現在の金融資産 | 2,000万円 |
| 現在の年齢 | 夫婦ともに35歳 |
| 子供 | 1人（0歳） |

FIRE目標は「年間支出×25倍」のいわゆる4%ルール。336万円×25 = **8,400万円**。

---

## ケース1：育休なし（ベースライン）

まず「育休を取らなかった場合」を計算しました。子どもはいるけど二人ともフルタイムで働き続ける想定。

**結果**
- FIRE到達年齢：**46歳**（今から11年後）
- FIRE時の推計資産：約1億円
- FIRE成功確率：**82%**

成功確率というのは、1,000通りの市場シナリオで「90歳まで資産が枯渇しない」確率です。1,000回シミュレーションして820回は大丈夫だった、ということ。

## ケース2：妻のみ育休12ヶ月

次に、妻が産前2ヶ月＋産後12ヶ月育休を取るケース。育休中は給付金のみ（給与の約67%）。育休明けから子が3歳になるまでは時短（月収30万円）。夫はフル就労のまま。

**結果**
- FIRE到達年齢：**49歳**（今から14年後）
- FIRE時の推計資産：約9,400万円
- FIRE成功確率：**80%**

**育休なしより3年遅れ。**

[ここにスクリーンショット：妻育休のシミュレーション結果]

## ケース3：夫婦ともに育休12ヶ月

「夫も育休を取ったら？」というケース。二人とも育休12ヶ月。育休明けは妻のみ時短（夫は通常勤務に戻る）。

**結果**
- FIRE到達年齢：**51歳**（今から16年後）
- FIRE時の推計資産：約9,000万円
- FIRE成功確率：**77%**

**育休なしより5年遅れ。妻育休だけのケースよりさらに2年遅れ。**

## 計算してみて、正直驚いた

妻だけ育休 → 3年遅れ。夫婦で育休 → 5年遅れ。

最初に結果を見たとき「あれ、思ってたより小さい」と感じました。育休1年で「10年遅れる」くらいのイメージを勝手に持っていたので。

なぜこの程度で済むのか、少し考えてみると：

**① 育休中も収入はゼロにならない**
最初の180日は給与の約67%が給付金として出る。全額なくなるわけじゃない。

**② 育休中は支出も減る**
保育料・外食・交通費・被服費あたりが下がる。収入が減ると支出も多少つられて下がるんですよね。

**③ 資産2,000万円がすでにある**
今ある資産が複利で育ち続けるので、後半でかなり取り返せる。序盤のハンデは思ったより小さい。

夫が育休12ヶ月取っても、妻だけとの差は「2年」。この数字を見て、私の中の計算が変わりました。

---

## 「確率5%の悪化」をどう見るか

82%から77%に下がった、というのが何を意味するか。

1,000通りの未来のうち、育休なしだと820通りで「90歳まで大丈夫」。夫婦育休だと770通り。

差の50通りは、リーマンショック級の暴落が育休期間と重なるような「最悪の組み合わせ」で出てくる。平均的なシナリオでは5年遅れ、悪いシナリオではそれ以上の影響が出る。

「確率5%分の悪化を許容するかどうか」は、数字では答えが出ない。
子どもとの時間に何を感じるか、という話になる。

うちはこの計算結果を見て、夫婦で育休を取ることにしました。

---

## 住宅費の扱いについて

今回は持ち家（ローン月10万円）前提です。賃貸だと支出が変わるため、FIRE到達年齢も変わります。

今回は「育休の影響」を見たかったので住宅費は固定しましたが、フル版のシミュレーターでは持ち家・賃貸どちらでも設定できます。

## まとめると

- 妻のみ育休12ヶ月 → FIREが**約3年遅れる**
- 夫婦で育休12ヶ月 → FIREが**約5年遅れる**（妻だけとの差は+2年）
- FIRE成功確率は**最大5%低下**（82% → 77%）

「育休を取るとFIREが大幅に遅れる」は、少なくともうちの数字では大げさでした。

確率で考えると、漠然とした不安が「じゃあ実際何年遅れるのか」という具体的な問いに変わります。その問いに答えが出たとき、ようやく判断できる気がします。
