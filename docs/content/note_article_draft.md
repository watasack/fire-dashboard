# 育休を取るとFIREが何年遅れるか、1,000通りの未来で計算してみた

#FIRE #共働きFIRE #育休 #子育てとお金 #モンテカルロ

※ FIRE到達年齢・資産額は特定の前提条件に基づくシミュレーション結果であり、将来の運用成果を保証するものではありません。FIRE成功確率はモンテカルロ推計値（参考）。

---

第一子が生まれる前、妻と二人で育休の話をしていました。

二人とも育休を取りたい気持ちはあった。でも「取ったら、FIREはどうなるんだろう」という不安がずっとあった。何年遅れるのか見当もつかない、というのが一番しんどかった。

漠然とした不安のまま決めたくなかったので、二人で計算してみることにしました。

---

## 前提条件

参考値として、うちの数字を載せます。収入や資産額が違っても、「育休がFIREに与える影響の傾向」は同じなので、比較の考え方は参考になると思います。

| 項目 | 数値 |
|------|------|
| 夫婦の手取り月収合計 | 70万円台 |
| 生活費 | 28万円/月 |
| 住宅ローン | 月10万円台（30年ローン） |
| 支出合計 | 38万円/月 |
| 現在の金融資産 | 2,000万円 |
| 現在の年齢 | 30代半ば |
| 子供 | 1人（0歳） |

FIRE達成の判定は、退職後の年金・教育費・住宅ローン・インフレ・税金などを含む実収支シミュレーションで「資産が尽きない最早の年齢」を算出する方式です。詳細な前提条件はシミュレーター内で確認できます。

---

## ケース1：育休なし（ベースライン）

まず影響の大きさを測るために、「育休を取らなかった場合」をベースラインとして計算しました。子どもはいるけど二人ともフルタイムで働き続ける想定です。

**結果**
- FIRE到達年齢：**46歳**（今から11年後）
- FIRE時の推計資産：約1億円
- FIRE成功確率：**82%**

成功確率というのは、1,000通りの市場シナリオで「90歳まで資産が枯渇しない」確率です。1,000回シミュレーションして820回は大丈夫だった、ということ。

## ケース2：妻のみ育休12ヶ月

次に、妻が産前2ヶ月＋産後12ヶ月育休を取るケース。育休中は給付金のみ（最初の180日は給与の約67%、以降は約50%）。育休明けから子が3歳になるまでは時短（月収30万円）。夫はフル就労のまま。

**結果**
- FIRE到達年齢：**49歳**（今から14年後）
- FIRE時の推計資産：約9,400万円
- FIRE成功確率：**80%**

**ベースラインより3年遅れ。**

## ケース3：夫婦ともに育休12ヶ月

「夫も育休を取ったら？」というケース。二人とも育休12ヶ月。育休明けは妻のみ時短（うちの場合は妻の職場のほうが時短制度を使いやすかったため。夫が時短のパターンもシミュレーターで計算できます）。

**結果**
- FIRE到達年齢：**51歳**（今から16年後）
- FIRE時の推計資産：約9,000万円
- FIRE成功確率：**77%**

**ベースラインより5年遅れ。妻だけのケースよりさらに2年遅れ。**

## 5年遅れ。でも10年じゃなかった

妻だけ育休で3年遅れ。夫婦で育休で5年遅れ。

二人で結果を見たとき、最初の感想は「あれ、思ってたより小さい」でした。なんとなく育休1年で「10年遅れる」くらいのイメージを持っていたので。

なぜこの程度で済むのか。一番効いているのは、すでにある資産2,000万円が複利で育ち続けること。育休中の1〜2年のブランクがあっても、後半でかなり取り返せます。序盤のハンデは思ったより小さい。

あとは育休中も収入がゼロにならないこと。最初の180日は給与の約67%が給付金として出ます。それに加えて、育休中は保育料・外食・交通費あたりの支出も減る。この辺は計算してみて初めて実感しました。

夫が育休12ヶ月取っても、妻だけとの差は「2年」。この数字を見て、二人とも「2年なら全然いいね」となりました。

---

## 「確率5%の差」をどう見るか

82%から77%に下がった、というのが何を意味するか。

1,000通りの未来のうち、育休なしだと820通りで「90歳まで大丈夫」。夫婦育休だと770通り。

差の50通りは、リーマンショック級の暴落が育休期間と重なるような「最悪の組み合わせ」で出てきます。平均的なシナリオでは5年遅れ、悪いシナリオではそれ以上の影響が出る。

ただ、この計算をしたことで「漠然と怖い」が「具体的に5%の差」に変わりました。何もわからないまま悩んでいた頃とは全然違って、安心して育休を選べた。うちはこの結果を見て、夫婦で育休を取ることに決めました。

---

## 住宅費の扱いについて

今回は持ち家（ローン月10万円）前提です。賃貸だと支出が変わるため、FIRE到達年齢も変わります。

今回は「育休の影響」を見たかったので住宅費は固定しましたが、フル版のシミュレーターでは持ち家・賃貸どちらでも設定できます。
