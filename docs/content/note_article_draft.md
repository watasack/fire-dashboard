# 育休を取るとFIREが何年遅れるか、1,000通りの未来で計算した（共働き・子供1人）

※ FIRE到達年齢・資産額は実際のシミュレーション結果。FIRE成功確率はフル版（モンテカルロ）の推計値（参考）。

---

## はじめに

「育休を取りたいけど、FIREが遅れるのが怖い」

共働きでFIREを目指している方なら、一度は考えたことがあると思います。
私も第一子の育休を前に、同じ不安を持っていました。

で、本気で計算しました。

1,000通りの未来を確率で計算したら、思ってたより面白い結果が出たので書きます。

---

## 前提条件

| 項目 | 数値 |
|------|------|
| 夫婦の手取り月収合計 | 75万円（夫35万・妻40万） |
| 生活費 | 28万円/月 |
| 住宅ローン | 10万円/月（2060年末完済） |
| 支出合計 | 38万円/月 |
| 現在の金融資産 | 2,000万円 |
| 現在の年齢 | 夫婦ともに35歳 |
| 子供 | 1人（0歳） |
| 雇用形態 | 夫婦ともに会社員 |

FIRE目標：年間支出（336万円）の25倍 = 8,400万円

---

## ケース1：育休なし（ベースライン）

夫婦ともにフル就労した場合のシミュレーション。子どもはいるが育休は取らない想定。

**結果**
- FIRE到達年齢：**46歳**（今から11年後）
- FIRE時推計資産：約1億円
- FIRE成功確率：**82%**（1,000通り中820通りで90歳まで資産が枯渇しない）

## ケース2：妻のみ育休12ヶ月

妻が産前2ヶ月・産後12ヶ月育休取得。育休中は給付金のみ（給与の約67%）。
育休復帰後：子が3歳になるまで時短勤務（時短月収30万円）。

夫は育休なし・フル就労継続。

**結果**
- FIRE到達年齢：**49歳**（今から14年後）
- FIRE時推計資産：約9,400万円
- FIRE成功確率：**80%**（フル版推計）

## ケース3：夫婦ともに育休12ヶ月

妻が産前2ヶ月・産後12ヶ月育休取得。**夫も育休12ヶ月取得。**
夫婦ともに育休中は給付金のみ（給与の約67%）。
育休復帰後：妻は子が3歳になるまで時短勤務。夫は時短なし。

**結果**
- FIRE到達年齢：**51歳**（今から16年後）
- FIRE時推計資産：約9,000万円
- FIRE成功確率：**77%**（フル版推計）

## 3ケースの比較

| 指標 | 育休なし | 妻のみ育休 | 夫婦育休 |
|------|---------|-----------|---------|
| FIRE到達年齢 | 46歳 | 49歳 | 51歳 |
| FIRE時資産 | 約1億円 | 約9,400万円 | 約9,000万円 |
| FIRE成功確率 | 82% | 80% | 77% |
| 育休なしとの差 | — | **+3年** | **+5年** |

---

## 考察

**妻だけ育休12ヶ月 → FIREは3年遅れ**
**夫婦で育休12ヶ月 → FIREは5年遅れ**

正直、思ってたより小さかったです。理由を分解すると：

1. **育休給付金があるので収入がゼロにならない**（最初の180日は給与の約67%）
2. **育休中は支出も多少下がる**（保育料・外食・交通費が減る）
3. **複利の力で、資産2,000万円からのスタートがあれば後半で取り返せる**

夫が育休を12ヶ月取っても、妻との差は2年（3年 → 5年）。
「夫が育休を長く取るとFIREが大幅に遅れる」というわけでもないのがポイントです。

---

## 成功確率が下がる意味

「82%から77%に下がった」というのは何を意味するか。

1,000通りの未来のうち：
- 育休なし → 820通りで90歳まで資産が枯渇しない
- 夫婦育休 → 770通りで90歳まで資産が枯渇しない

差の50通りは、**景気の悪い年が重なる最悪シナリオ**で影響が出る。
平均シナリオでは5年遅れ、悪いシナリオではさらに影響が大きい。

「確率5%分の悪化をどう評価するか」は、家族の価値観の問題です。

---

## 住宅費の扱いについて

今回は持ち家（住宅ローン月10万円）を前提にしています。
賃貸（月15〜20万円）の場合は支出が変わるため、FIRE到達年齢も変わります。

フル版シミュレーターでは持ち家・賃貸どちらでも設定できます。

**フル版（有料・買い切り）では：**

- 雇用形態（会社員／個人事業主／専業主婦）ごとの年金・昇給の違いを自動反映
- 子どもの人数と誕生日を入力するだけで教育費を自動計算（最大4人）
- 1,000通りのシミュレーションでFIRE成功確率を算出
- 1,000通りの資産推移をグラフで見える化

## まとめ

- 妻のみ育休12ヶ月でFIREが**約3年遅れる**
- 夫婦で育休12ヶ月でもFIREが**約5年遅れる**（妻との差は+2年）
- FIRE成功確率は**最大5%低下**（82% → 77%）
- 「育休を取るとFIREが大幅に遅れる」は思ったより大げさだった

確率で考えると、育休の「コスト」がはっきり見えて、判断しやすくなります。
