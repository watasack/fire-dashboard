# 育休12ヶ月取ったら、FIREが何年遅れるか本気で計算した（共働き・子供1人）

※ FIRE到達年齢・資産額は実際のシミュレーション結果。FIRE成功確率はフル版（モンテカルロ）の推計値（参考）。

---

## はじめに

「育休を取りたいけど、FIREが遅れるのが怖い」

共働きでFIREを目指している方なら、一度は考えたことがあると思います。
私も第一子の育休を前に、同じ不安を持っていました。

で、本気で計算しました。

数理モデリングをやっている手前、Excelじゃなくてモンテカルロシミュレーションで。
1,000通りの未来を計算して、確率で考えてみたら、思ってたより面白い結果が出たので書きます。

---

## 前提条件

| 項目 | 数値 |
|------|------|
| 夫婦の手取り年収合計 | 900万円 |
| 月間支出 | 28万円（年336万円） |
| 現在の金融資産 | 2,000万円 |
| 現在の年齢 | 35歳 |
| 子供 | 1人（0歳） |

FIRE目標：年間支出の25倍 = 8,400万円（4%ルールベース）

---

## ケース1：育休なし（ベースライン）

育休を取らずに夫婦ともにフル就労した場合のシミュレーション。

**結果**
- FIRE到達年齢：**46歳**（今から11年後）
- FIRE時推計資産：約1億円
- FIRE成功確率：**82%**（フル版推計・1,000回中820回、資産が枯渇しない）

## ケース2：産休2ヶ月 + 育休12ヶ月あり

妻が産前2ヶ月・産後12ヶ月育休取得。夫が育休1ヶ月取得。育休中は給付金のみ受け取り（妻：給与の67%、夫：給与の67%）。

育休復帰後：妻は子が3歳になるまで時短勤務（給与の75%）。

**結果**
- FIRE到達年齢：**50歳**（今から15年後）
- FIRE時推計資産：約9,600万円
- FIRE成功確率：**78%**（フル版推計）

## 差分の考察

| 指標 | 育休なし | 育休あり | 差分 |
|------|---------|---------|------|
| FIRE到達年齢 | 46歳 | 50歳 | **+4年** |
| FIRE時資産 | 約1億円 | 約9,600万円 | -約400万円 |
| FIRE成功確率 | 82% | 78% | -4% |

**育休12ヶ月でFIREは4年遅れる。**

これ、正直思ってたより小さかったです。

理由を分解すると：

1. **育休給付金があるので収入がゼロにならない**（夫は給与の67%、最初の180日）
2. **育休中は支出も多少下がる**（外食・交通費・保育料がなくなる）
3. **複利の力で、若いうちの4年遅れは後半で取り返せる部分がある**

もちろん、育休中の収入減が大きい場合（妻が個人事業主で給付金なし）だと遅れはもっと大きくなります。条件によって結果は変わります。

---

## 成功確率が4%下がる意味

「82%から78%に下がった」というのは、どういう意味か。

1,000通りの未来のうち：
- 育休なし → 820通りで資産が枯渇しない
- 育休あり → 780通りで資産が枯渇しない

つまり、リターンが悪い年が重なる最悪シナリオでは、育休の影響が出てくる。
平均シナリオでは4年遅れで、リスクシナリオではもう少し影響が大きい。

## まとめ

- 育休12ヶ月でFIREが**約4年遅れる**（条件次第で変わる）

「育休を取るとFIREが大幅に遅れる」という漠然とした不安は、計算してみると思ったより小さかった。でも「全く影響がない」わけでもない。
