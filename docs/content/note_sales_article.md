# 共働き・子育て世帯のFIREシミュレーター【フル版】育休・産休・時短を正確に反映して、1,000通りの未来を計算する

---

## 「育休中の計算ができない」問題

FIREを本気で目指している共働きの方が、一度は直面する壁があります。

「育休を取ったら、FIREはどうなるんだろう」

世の中のFIREシミュレーターのほとんどは、収入が一定という前提で動いています。育休中の給付金、時短勤務への移行、保育料の発生、子どもが増えることによる支出の変化——こういう「共働き・子育て世帯のリアル」を、まとめて計算できるツールはほぼ存在しません。

このシミュレーターは、その問題を解決するために作りました。

作者自身が共働き・子育て世帯でFIREを計画しており、「自分の状況を正確に計算できるツールがない」という課題を感じて3年かけて作ったものです。育休取得の決断、支出の見直し、NISAの積立額の設定——実際に自分の意思決定に使ってきたツールを、同じ状況の方に使ってもらいたいと思い公開しました。

## このシミュレーターでできること

**育休・産休・時短勤務を月単位で設定できる**

産前○ヶ月・産後○ヶ月・育休○ヶ月を個別に入力できます。育休中の月収（給付金）も設定できるので、「給付金込みでいくら収入が下がるか」を正確に計算します。時短勤務の期間と時短中の月収も設定可能。夫・妻それぞれ独立して設定できます。

「妻だけ育休12ヶ月 vs 夫婦ともに12ヶ月」のような比較が、実際の数字でできます。

**子どもの教育費を自動で計算する**

誕生日と教育方針（公立中心・やや手厚め・私立重視）を入力するだけで、幼稚園から大学卒業までの費用を自動で積算します。最大4人まで対応。「第2子が生まれたときにどう変わるか」も計算できます。

**1,000通りの未来でFIRE成功確率を出す**

市場のリターンは毎年異なります。「平均的なケース」だけを見ていると、リーマンショック級の暴落が重なったときのリスクが見えません。

このシミュレーターは1,000通りの市場シナリオで計算し、「何%のシナリオで90歳まで資産が枯渇しないか」をFIRE成功確率として表示します。最悪ケースで資産がどこまで減るかも確認できます。

**住宅費・雇用形態・NISA・iDeCoを全部まとめて計算**

持ち家（ローン月額・完済年）か賃貸（月額）を選んで入力。会社員か個人事業主かで年金・昇給の計算が変わります。NISA・iDeCoの残高と積立額も反映します。

## 無料ツールとの違い

| | 一般的な無料ツール | このシミュレーター |
|---|---|---|
| 育休・産休 | 非対応 | 月単位で設定可能 |
| 時短勤務 | 非対応 | 終了年齢まで設定可能 |
| 教育費 | 概算のみ | 誕生日から自動計算 |
| FIRE成功確率 | 非対応 | 1,000通りで算出 |
| 最悪ケース分析 | 非対応 | 資産枯渇シナリオを表示 |
| 夫婦の個別設定 | 非対応か簡易 | 雇用形態・収入・育休を個別設定 |
