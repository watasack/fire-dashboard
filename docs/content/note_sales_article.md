# 共働き・子育て世帯のFIREシミュレーター【フル版】育休・産休・時短を正確に反映して、1,000通りの未来を計算する

---

## このツールで解決できること

「FIREを目指しているけど、育休中の計算ができない」

共働きでFIREを本気で計算しようとしたとき、こう思ったことはありませんか。

世の中のFIREシミュレーターのほとんどは、収入が一定という前提で動いています。
育休・産休・時短勤務・子どもの教育費——これらを全部まとめて計算できるツールは、ほとんど存在しません。

このツールは、「共働き・子育て世帯のFIREを正確に計算する」ために、エンジニアが3年かけて自分用に作り、公開したものです。

---

## できること

**育休・産休・時短勤務の完全反映**
- 産前○ヶ月・産後○ヶ月・育休○ヶ月を個別に設定
- 育休中の月収（給付金）を入力して正確に計算
- 時短勤務の終了年齢・時短中の月収を設定
- 夫・妻それぞれ独立して設定可能

**子どもの教育費を自動計算**
- 誕生日を入力するだけで、幼稚園〜大学までの費用を自動積算
- 教育方針（公立中心／やや手厚め／私立重視）を選択するだけ
- 最大4人まで設定可能

**住宅費の正確な反映**
- 持ち家：ローン月額と完済年を入力
- 賃貸：月額家賃を入力

**雇用形態に応じた年金・昇給の自動計算**
- 会社員：厚生年金・毎年の昇給を自動反映
- 個人事業主：国民年金・収入固定で計算
- 夫婦で異なる雇用形態も設定可能

**1,000通りのモンテカルロシミュレーション**
- リーマンショック級の暴落を含む1,000通りの市場シナリオで検証
- 「FIRE成功確率○%」として数値で表示
- 最悪ケース（資産が枯渇するシナリオ）の分析

**FIRE後の柔軟な設定**
- サイドFIRE（FIRE後も副収入あり）の設定
- 取り崩し方法の選択（固定額／定率／ガードレール戦略）
- NISA・iDeCoの残高・積立額を反映

## 無料ツールとの違い

| | 無料ツール（一般的） | このツール |
|---|---|---|
| 育休・産休 | 非対応 | ✅ 月単位で設定 |
| 時短勤務 | 非対応 | ✅ 終了年齢まで設定 |
| 教育費 | 概算のみ | ✅ 誕生日から自動計算 |
| FIRE成功確率 | 非対応 | ✅ 1,000回シミュレーション |
| 最悪ケース分析 | 非対応 | ✅ 資産枯渇シナリオを表示 |
| 夫婦別設定 | 非対応か簡易 | ✅ 雇用形態・収入・育休を個別設定 |

## 作った人について

宇宙・航空系エンジニア。本業で機械学習・数理モデリングを使っています。

自分自身が共働き・子育て世帯でFIREを計画しており、「自分の状況を正確に計算できるツールがない」という課題を感じて作りました。このツールを使って、実際に育休取得・支出見直し・NISA積立額の意思決定をしています。

金融の専門家ではありません。「数理モデリングの専門家が自分用に作ったシミュレーター」として使ってください。
