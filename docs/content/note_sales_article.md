## このシミュレーターでできること

<!-- 📷 画像: result_top.png（ツール全体のファーストビュー: KPI + チャート） -->

**育休・産休・時短勤務を月単位で設定できる**

産前○ヶ月・産後○ヶ月・育休○ヶ月を個別に入力できます。育休中の月収（給付金）も設定できるので、「給付金込みでいくら収入が下がるか」を正確に計算します。時短勤務の期間と時短中の月収も設定可能。夫・妻それぞれ独立して設定できます。

「妻だけ育休12ヶ月 vs 夫婦ともに12ヶ月」のような比較が、自分の家計の数字でできます。

**子どもの教育費を自動で計算する**

誕生日と教育方針（公立中心・やや手厚め・私立重視）を入力するだけで、幼稚園から大学卒業までの費用を自動で積算します。最大4人まで対応。「第2子が生まれたときにどう変わるか」も計算できます。

**1,000通りの未来でFIRE成功確率を出す**

市場のリターンは毎年異なります。「平均的なケース」だけを見ていると、リーマンショック級の暴落が重なったときのリスクが見えません。

このシミュレーターは1,000通りの市場シナリオで計算し、「何%のシナリオで90歳まで資産が枯渇しないか」を推計値として表示します（特定の前提条件に基づく参考値であり、将来の成果を保証するものではありません）。最悪ケースで資産がどこまで減るかも確認できます。

**住宅費・雇用形態・NISA・iDeCoを全部まとめて計算**

持ち家（ローン月額・完済年）か賃貸（月額）を選んで入力。会社員か個人事業主かで年金・昇給の計算が変わります。NISA・iDeCoの残高と積立額も反映します。

## 無料ツールとの違い

| | 一般的な無料ツール | このシミュレーター |
|---|---|---|
| 育休・産休 | 非対応 | 月単位で設定可能 |
| 時短勤務 | 非対応 | 終了年齢まで設定可能 |
| 教育費 | 概算のみ | 誕生日から自動計算 |
| FIRE成功確率 | 非対応 | 1,000通りで算出 |
| 最悪ケース分析 | 非対応 | 資産枯渇シナリオを表示 |
| 夫婦の個別設定 | 非対応か簡易 | 雇用形態・収入・育休を個別設定 |

## 実際の計算例

別記事で、うちの家計の数字を使って「育休でFIREが何年遅れるか」を計算しています。

- 育休なし → FIRE 46歳（成功率82%）
- 妻のみ育休12ヶ月 → FIRE 49歳（成功率80%）
- 夫婦で育休12ヶ月 → FIRE 51歳（成功率77%）
