# UI説明文レビュー用ドキュメント

## 使い方
各項目について **現状 / パターンA / パターンB** の3案を掲載しています。
- **現状**: 現在のUI表示テキスト
- **パターンA**: 対象ペルソナ向け最適化（共働き・子育て世帯でFIREを考える人・金融専門家でない）
- **パターンB**: さらに分かりやすい版（具体例・入力ガイド付き）

各行の「採用」列に A / B / 現状 を記入してください。

---

## 対象ペルソナ
- 共働きで子育て中、FIREを本気で考えている
- 金融の専門知識はない（FPではない）
- 育休・時短・教育費など変数が多くて困っている
- 「で、実際いくら必要なの？」が知りたい人
