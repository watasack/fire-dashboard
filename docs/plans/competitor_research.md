### 計算ロジック詳細

#### 税金計算（累進課税）
- 本人控除: 43万 + 38万（年齢条件） + 38万×扶養人数
- 配偶者控除: 43万 + 33万（年齢条件） + 33万×扶養人数
- 配偶者所得が扶養閾値以下の場合は計算除外

#### 社会保険料
- 健康保険: 9.98%（40歳以上は介護保険分が加算）
- 年金: 18.3%（月額65万円の上限あり）
- 事業主折半のため実質は各半額

#### 資産推移
```
資産(t+1) = 資産(t) × (1 + リスク比率 × 5%) + 月積立 × 12
```

#### 年金
- 一律180万円/年で自動推定（固定値）
- 65歳以降の収入として計算に組み込む

#### 教育費（子どもの年齢と教育方針に基づく自動計算）

| 学校段階 | 公立（万円/年） | 私立（万円/年） |
|----------|--------------|--------------|
| 幼稚園 | 30 | 50 |
| 小学校 | 35 | 150 |
| 中学校 | 50 | 150 |
| 高校 | 50 | 100 |
| 大学 | 120 | 150 |

インフレ調整: 年2%で増額

### モンテカルロシミュレーションの実装詳細

| 項目 | 内容 |
|------|------|
| 試行回数 | **3,000回** |
| 確率分布 | **Student-t分布（自由度12）** |
| 分布の特徴 | 正規分布より裾が重い → 大暴落の発生確率をより現実的に再現 |
| マルチアセット | QQQ（NASDAQ100）・GLD（金ETF）・EDV（超長期国債）・BND（総合債券）の4資産 |
| 資産間相関 | **コレスキー分解**で相関行列から連動性を再現 |

---

### 取り崩し戦略（3種類）

| 戦略 | 内容 |
|------|------|
| 金額指定 | 毎年固定額を取り崩す（従来の4%ルール型） |
| %指定（トリニティ準拠） | 資産残高の一定割合を毎年取り崩す |
| ガードレール＋U字型支出 | 資産が減れば引き出しを下げ、増えれば上げる動的調整。年齢ごとのU字型支出カーブ（若年期・老年期は多く、中間期は少ない）を反映 |

---

### 日本固有の税・社会保険の計算

- 金融所得に対する **20.315%課税**（含み益割合に応じて計算）
- 将来的な金融所得への社会保険料上乗せ（**+6%想定**）
- **国民健康保険料**（前年所得ベースで自動計算）
- **住民税**（前年所得ベースで自動計算）
- **国民年金保険料**
- **FIRE初年度の国保スパイク**を個別計算（前年の給与所得が高い場合に初年度コストが大きく跳ね上がる）

| フロア割れからの復帰率 | 一度下がった後に回復できる割合 |
| 復帰までの年数（中央値） | 下落後の回復期間 |
