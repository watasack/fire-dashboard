### 計算ロジック詳細

#### 税金計算（累進課税）
- 本人控除: 43万 + 38万（年齢条件） + 38万×扶養人数
- 配偶者控除: 43万 + 33万（年齢条件） + 33万×扶養人数
- 配偶者所得が扶養閾値以下の場合は計算除外

#### 社会保険料
- 健康保険: 9.98%（40歳以上は介護保険分が加算）
- 年金: 18.3%（月額65万円の上限あり）
- 事業主折半のため実質は各半額

#### 資産推移
```
資産(t+1) = 資産(t) × (1 + リスク比率 × 5%) + 月積立 × 12
```

#### 年金
- 一律180万円/年で自動推定（固定値）
- 65歳以降の収入として計算に組み込む

#### 教育費（子どもの年齢と教育方針に基づく自動計算）

| 学校段階 | 公立（万円/年） | 私立（万円/年） |
|----------|--------------|--------------|
| 幼稚園 | 30 | 50 |
| 小学校 | 35 | 150 |
| 中学校 | 50 | 150 |
| 高校 | 50 | 100 |
| 大学 | 120 | 150 |

インフレ調整: 年2%で増額

- **住民税**（前年所得ベースで自動計算）
- **国民年金保険料**
- **FIRE初年度の国保スパイク**を個別計算（前年の給与所得が高い場合に初年度コストが大きく跳ね上がる）

### モンテカルロシミュレーションの実装詳細

| 項目 | 内容 |
|------|------|
| 試行回数 | **10,000回** |
| 手法 | **ブートストラップ法**（正規分布ではなく実績データからの無作為抽出） |
| データソース | 1971〜2022年の実際の過去データ（約52年分） |
| データ内訳 | 日経平均・S&P500・日米国債流通利回り・USD/JPY・インフレ率 |

| 相関の保持 | **各年のシミュレーションで株・債券・為替・インフレ率を必ず同じ年のデータで使用**（資産クラス間の歴史的相関を自動保持） |

**ブートストラップ法の特長**: 正規分布の仮定が不要。1971〜2022年に実際に起きた暴落（オイルショック・リーマンショック等）をそのまま再現できる。資産クラス間の相関も歴史的に正確。

---

### 毎年のシミュレーションフロー

1. インフレ調整後の年間生活費を引き出す
2. 予定配分でリバランス（リバランス時の税・為替コストは非考慮）
3. 運用による年間収益・損失を加算
4. 運用以外の収益・損失を加算
5. 資産総額を円建てで記録

---

### 出力・グラフ

- **パーセンタイル資産推移グラフ**: 1パーセンタイル・50パーセンタイル等、複数パーセンタイルの資産額を年別に表示
